# Tech Context

This project is a static website generated by Jekyll.

- **Framework:** Jekyll
- **Theme:** `git-wiki-theme` (Internalized/Forked Local Copy)
  - **Origin:** Forked from `Drassil/git-wiki-theme`.
  - **Modifications:** 
    - Native Catppuccin color scheme integration via SCSS.
    - **Strict Color Schemes:** Enforced Latte (Light) and Macchiato (Dark) only.
    - Removal of `darkmode.js`.
    - Fonts via Google Fonts (Noto Sans).
  - **Components:** Modular components (`action_btn`, `copyrights`, `lists`, `logo`, `search`, `toc`) located in `_includes/git-wiki`.
  - **Search:** Client-side JS search using `simple-jekyll-search` (via CDN).
  - **Styling Architecture:**
    - `_sass/_catppuccin.scss`: Defines CSS variables for Latte and Macchiato.
    - `_includes/custom_style.html`: Handles theme logic (mapping `--ctp-main-*` variables).
    - `assets/css/git-wiki-style.scss`: Main entry point compiling to CSS.
- **Language:** Ruby (for Jekyll)
- **Content:** Markdown (`.md`) files
- **Package Management:** Bundler (via `Gemfile`)
- **Deployment:** Compatible with GitHub Pages and GitLab Pages.
- **Local Development:** Docker and `docker-compose` are used to run a local Jekyll server.
- **CI/CD:** GitLab CI (`.gitlab-ci.yml`) configured.
