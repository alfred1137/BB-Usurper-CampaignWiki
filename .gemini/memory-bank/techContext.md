# Tech Context

This project is a static website generated by Jekyll.

- **Framework:** Jekyll
- **Theme:** `Drassil/git-wiki-theme` (used as a remote theme)
- **Language:** Ruby (for Jekyll)
- **Content:** Markdown (`.md`) files
- **Package Management:** Bundler (via `Gemfile`)
- **Deployment:** The setup is compatible with GitHub Pages and GitLab Pages.
- **Local Development:** Docker and `docker-compose` are used to run a local Jekyll server. There are separate Docker configurations for GitHub (`Dockerfile.github`, `Gemfile.github`) and GitLab (`Dockerfile.gitlab`, `Gemfile.gitlab`).
- **CI/CD:** A `.gitlab-ci.yml` is present, suggesting GitLab CI is used for continuous integration and deployment.
